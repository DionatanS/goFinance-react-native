# goFinances

Este projeto é uma simulação de uma aplicação de finanças mobile utilizada para estudos praticos de react-native com expo, com a intenção de agregar mais conhecimentos desse framework.

## Executar o projeto:
- **`yarn install`**
- **`yarn start`**
